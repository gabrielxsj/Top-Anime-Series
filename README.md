Funcionalidades Principais:

    -Listagem dos animes mais bem avaliados (Top Animes)
    -Carrega automaticamente uma lista paginada dos animes mais populares e bem ranqueados
    -Busca por nome
    -Filtro alfabético (A–Z)
    -Paginação e Scroll Infinito
    -Página de detalhes do anime
    -Lista completa de personagens
    -Biografia completa
    -Duladores com foto e idioma

Tecnologias Utilizadas:

    -React.js
    -Hooks (useState, useEffect)
    -CSS customizado
    -Jikan API v4 (para dados de animes e personagens)
